# CodeAlpha Hangman Game 🎮

## 📌 Task: Hangman Game (Python Internship)

This is a simple **text-based Hangman game** built using Python.  
The player has to guess a randomly chosen word by entering one letter at a time.

---

## 🧠 Features
- Randomly picks a word from a predefined list.
- Allows up to 6 incorrect guesses.
- Displays the progress of the word after each guess.
- Console-based user interaction.

---

## 💻 Tech Stack
- Python
- Built-in modules: `random`

---

## 🔁 How to Run
1. Clone or download this repository.
2. Open terminal and navigate to the project folder.
3. Run the script:
```bash
python hangman.py
