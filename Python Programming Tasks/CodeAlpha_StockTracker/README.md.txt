# CodeAlpha Stock Portfolio Tracker 📊

## 📌 Task: Stock Portfolio Tracker (Python Internship)

This project calculates the total investment value based on user inputs and predefined stock prices.  
It's a console-based application built using core Python.

---

## 🧠 Features
- Accepts stock quantities for selected companies.
- Calculates total investment value.
- Saves the summary in a `.txt` file.

---

## 💻 Tech Stack
- Python
- Built-in functions: `input()`, `open()`, `dict`

---

## 🔁 How to Run
1. Clone or download this repository.
2. Open terminal and navigate to the project folder.
3. Run the script:
```bash
python stock_tracker.py
